Common Symbols and what they mean



Physics has an overwhleming amount of symbols and its easy to get lost in what they mean. So this page is going to teach you what some of the common symbols are and wha they mean.

m = mass. This is one of the most basic symbols in physics. it is used to measure the mass of an object.

v = velocity. V is used to measure the velocity of an object and is calculated by using Distance/Time.

t = time. T is used to measure the time of anything and is present is serveral equations.

a = acceleration. A is calculated by using Veclocity/Time.

Fw = weight force. Fw is used to measure the weight force of an object it is calculated by using mass x gravity.

p = momentum. P is calculated using mass x velocity

g = Gravity. Gravity is always 9.8m/s^2
